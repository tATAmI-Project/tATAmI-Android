Compiling
  - be sure to have JDK 6 installed, with JRE 6 configured in Eclipse -> Window -> Preferences -> Java -> Installed JREs, with the name "jre6".
  (this is because Android development supports only Java 6)
  - ant-run importer.xml
  - refresh project

  
Running
  - run project as Android application
  - run forward-port.bat
  - start the platform (tATAmI-PC)
  - connect the application to the platform
